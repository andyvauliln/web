## USEFULLL

### Trigger a github action
```bash curl -X POST \
  -H "Accept: application/vnd.github.v3+json" \
  -H "Authorization: token YOUR_GITHUB_TOKEN" \
  https://api.github.com/repos/USERNAME/REPOSITORY/dispatches \
  -d '{"event_type":"run-script"}'
```
